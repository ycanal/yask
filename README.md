Yet Another Starter Kit
---

The goal of this project is to provide the boilerplate for:
* react
* redux
* react-router
* bootstrap 3 / less

Usage
---


Setup
---

```
npm install
```



Start the development server with this command:

```
npm start
```



Build for development environment
---

```
npm run build:dev
```

Build for production environment
---

```
npm run build:prod
```
